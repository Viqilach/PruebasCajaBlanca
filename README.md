# Viajero Intergaláctico - Pruebas de Caja Blanca

## Descripción
Juego de tablero galáctico implementado en Java.
Se analiza su complejidad ciclomática mediante grafos de flujo de control.

## Tecnologías utilizadas
- Java
- Eclipse
- Git & GitHub

## Cómo hacer el deploy (ejecutar el proyecto)
1. Clonar el repositorio:
   git clone https://github.com/Viqilach/PruebasCajaBlanca.git
2. Abrir Eclipse → File → Open Projects from File System
3. Seleccionar la carpeta del proyecto
4. Finish
